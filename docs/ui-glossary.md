# 界面术语表

描述需求时的统一用词。中文名 / 代码名对照，附主要实现文件，方便直接定位。

## 整体布局（5 个区域）

| 中文名 | 代码名 | 位置与说明 | 主要文件 |
|--------|--------|-----------|----------|
| 顶部工具栏 | header / action row | 最上一行：撤销/重做、文本、色块、导入素材、保存等 | `apps/editor/src/App.tsx` |
| 画布 | canvas | 中间黑色大区域；居中的视频画面叫**合成**（composition） | `apps/editor/src/canvas/CanvasView.tsx` |
| 检查器 | Inspector | 右侧面板，随选中对象变化 | `apps/editor/src/inspector/Inspector.tsx` |
| 播放条 | playback bar | 画布下方一细条：播放/暂停、时间码、循环、全屏 | `apps/editor/src/playback/PlaybackBar.tsx` |
| 时间线 | timeline | 最底部的多轨面板，上边缘可拖动调高度 | `apps/editor/src/timeline/TimelinePanel.tsx` |

## 时间线

| 中文名 | 代码名 | 说明 |
|--------|--------|------|
| 轨道 | track | 横向的一行（Track 1、Track 2…） |
| 轨道头 | track header | 轨道左侧：名字 + 眼睛（隐藏）+ 喇叭（静音） |
| 块 / 条目 | item / clip | 轨道上的彩色长条；按类型叫**视频块 / 音频块 / 图片块 / GIF 块 / 文本块 / 色块 / 字幕块** |
| 播放头 | playhead | 红色竖线，当前播放位置，可拖动 |
| 标尺 | ruler | 顶部时间刻度（0:01、0:02…），点击/拖动可跳转 |
| 修剪手柄 | trim handle | 块左右边缘，拖动伸缩时长（Alt+拖 = 滚动编辑） |
| 淡入淡出手柄 | fade handle | 块顶部两角小圆点，水平拖出渐入/渐出 |
| 音量线 | volume line | 音/视频块上的黄色横线，上下拖调音量（上=100%） |
| 波形 | waveform | 音频块（及有声视频块底部）的声音波纹 |
| 胶片缩略图 | filmstrip | 视频块上的一排预览帧 |
| 磁吸开关 | snapping toggle | 工具栏磁铁按钮，开关吸附对齐（Shift+M） |
| 剪刀按钮 | split button | 在播放头处分割；没选中时切播放头下所有块（S） |
| 斜纹指示器 | max trim indicator | 拖修剪手柄时的斜纹区，表示媒体还能延长的最大范围 |
| 拖放指示 | drop indicator | 拖文件到时间线时的蓝色落点竖线 + 轨道高亮 |
| 缩放滑杆 | zoom slider | 时间线工具栏右侧，调节时间线横向缩放 |

主要文件：`timeline/TimelinePanel.tsx`（面板/拖拽）、`timeline/ItemBlock.tsx`（块本体）、`timeline/ops.ts`（移动/修剪/分割等纯逻辑）。

## 画布

| 中文名 | 代码名 | 说明 |
|--------|--------|------|
| 选择框 | selection outline | 选中元素的蓝色边框 |
| 缩放手柄 | resize handle | 选择框四角/四边 8 个小点（Shift 解除等比锁定） |
| 吸附辅助线 | snap guide | 拖动时的红色对齐参考线（画布边/中心/其他元素） |
| 裁剪模式 | crop mode | 双击视频/图片进入；半透明全图叫**裁剪背景** |
| 行内文本编辑 | inline text edit | 双击文本块直接在画布上改字 |
| 框选 | marquee | 空白处拖矩形多选（时间线上同样支持） |
| 缩放控件 | zoom widget | 画布右上角 − 百分比 ＋（Cmd/触控板捏合滚轮、+/-/0 快捷键） |
| 绘制色块模式 | draw solid mode | 点工具栏「色块」进入，十字光标拖出矩形 |
| 右键菜单 | context menu | 置于顶层 / 置于底层 |

主要文件：`canvas/SelectionOverlay.tsx`、`canvas/CropOverlay.tsx`、`canvas/TextEditOverlay.tsx`、`canvas/DrawSolidOverlay.tsx`。

## 检查器分区

| 分区名 | 何时出现 | 内容 |
|--------|----------|------|
| 合成设置 | 没选中任何块 | 宽/高、交换尺寸 |
| 渲染区 | 没选中任何块 | 格式（MP4/WebM）、渲染按钮、进度、下载 |
| 位置/尺寸/旋转/不透明度/圆角/淡入淡出 | 选中任意块 | 数字输入（可按住拖动改值）|
| 对齐 | 选中任意块 | 左/中/右/上/中/下 6 个按钮 |
| 源信息 | 选中素材类块 | 文件名、大小、类型、时长、上传状态（含进度%） |
| 裁剪 | 选中视频/图片 | 进入裁剪、重置、X/Y/宽/高数值 |
| 文本样式 | 选中文本块 | 字体选择器（搜索+悬停预览）、字重、字号、颜色、行高、字间距、对齐、方向、文字背景 |
| 媒体 | 选中音/视频/GIF | 音量（dB）、倍速、静音 |
| 字幕面板 | 选中字幕块 | 样式设置 + 逐词修正列表；音/视频块上有「生成字幕」按钮 |

## 顶部工具栏与播放条

| 中文名 | 说明 |
|--------|------|
| 文本 / 色块 / 导入素材 | 三个创建工具（色块是拖拽绘制模式） |
| 保存 | 手动保存到浏览器（Cmd+S），未保存时按钮带黄点 |
| 下载状态 / 导入状态 | 把工程导出为 .json / 从 .json 恢复 |
| 清理素材 | 永久删除已移除的远端素材（弹确认对话框） |
| 状态徽章 | 上传中/转录中的进度提示 |
| 跳到开头 / 跳到结尾 / 播放暂停 / 时间码 / 静音 / 循环 / 全屏 | 播放条从左到右 |
