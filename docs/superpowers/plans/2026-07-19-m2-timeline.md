# M2：时间轴核心 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:executing-plans。

**Goal:** 完整时间轴编辑体验：轨道/条块/播放头/跨轨拖动/修剪/分割/吸附/框选/缩放/轨道操作。

**规格来源:** specs/2026-07-19-remotion-editor-design.md §5.2、M2 验收

## Global Constraints

- 帧为时间真源；像素 = 帧 × pxPerFrame（时间轴缩放）
- 同轨道内 Item 时间区间不得重叠：拖放落点冲突时放弃该次落置（官方行为：非法位置回弹）
- 拖拽/修剪过程 commit:false，松手 commitPending
- 时间轴纯逻辑全部进 `timeline/ops.ts`（可单测），组件只做事件→坐标换算

### Task 1: 时间轴纯逻辑 ops（TDD）

**Files:** `apps/editor/src/timeline/ops.ts`；Test: `apps/editor/src/timeline/__tests__/ops.test.ts`

**Interfaces (Produces):**
```ts
/** 区间重叠检测：同轨道内 [from, from+dur) 与既有 items（排除 ignoreIds）是否冲突 */
export const hasOverlap = (state: UndoableState, trackId: string, from: number, durationInFrames: number, ignoreIds: string[]) => boolean;
/** 批量移动（跨轨/跨时间）；任一目标位置冲突或 from<0 则返回原 state（引用不变） */
export const moveItems = (state: UndoableState, moves: { id: string; trackId: string; from: number }[]) => UndoableState;
/** 修剪：edge='start' 改 from+duration（媒体项同步 trimBefore，含素材上限钳制）；edge='end' 改 duration（媒体项按剩余素材长度钳制）*/
export const trimItem = (state: UndoableState, id: string, edge: 'start' | 'end', deltaFrames: number) => UndoableState;
/** 播放头处分割选中项（或全部命中项）：一分为二，媒体项右半 trimBefore 顺延 */
export const splitItemsAtFrame = (state: UndoableState, frame: number, itemIds: string[]) => UndoableState;
/** 吸附：候选=0、播放头、所有 item 端点；容差内返回吸附后的帧 */
export const snapFrame = (state: UndoableState, frame: number, toleranceFrames: number, opts?: { playheadFrame?: number; ignoreIds?: string[] }) => number;
/** 轨道操作 */
export const addTrack = (state: UndoableState, index: number) => { state: UndoableState; trackId: string };
export const removeEmptyTracks = (state: UndoableState) => UndoableState;
/** 媒体项素材总帧数（按 playbackRate 换算）；非媒体项返回 null */
export const maxItemDurationInFrames = (state: UndoableState, id: string) => number | null;
```

**测试用例:**
- hasOverlap：相邻不算重叠（[0,30)+[30,60)）；交叠算；ignoreIds 生效
- moveItems：合法移动生效；冲突回弹（引用不变）；跨轨道移动 trackId 变化；负 from 回弹；多项一起移动内部互不冲突（ignore 集合为整体）
- trimItem start：+10 ⇒ from+10、dur−10、媒体 trimBefore+10；dur 钳到 ≥1；trimBefore 不小于 0（图片/文本无 trimBefore 仍可修剪）
- trimItem end：媒体项不能超过素材剩余长度；文本/纯色无上限
- splitItemsAtFrame：帧在项内 ⇒ 两段总时长不变、右半 from=frame、媒体右半 trimBefore 顺延；帧在项外 ⇒ 不变
- snapFrame：容差内吸到端点/播放头/0；容差外原样返回；ignoreIds 排除自身端点
- maxItemDurationInFrames：video 10s@30fps rate=2 ⇒ 150 帧；text ⇒ null

### Task 3: 时间轴交互（拖动/修剪/分割/吸附/框选/轨道操作）

**Files:** `apps/editor/src/timeline/ItemBlock.tsx`（拖动+修剪手柄）, `TimelinePanel.tsx`（框选、新轨道落点）, `timeline/drag.ts`（换算 helpers）；Modify: `shortcuts/useShortcuts.ts`（S 分割在播放头、Shift+M 吸附开关）

- 条块拖动：横向改 from、纵向跨轨道（高亮目标轨道）；多选联动；Snap 吸附端点；冲突落点回弹（moveItems 语义）；拖到最上/最下边缘 ⇒ 新建轨道落入
- 修剪：条块两端手柄，拖动调用 trimItem（逐帧 commit:false）；媒体项显示最大可修剪范围提示（手柄拖到极限时边缘变红）
- 分割：S 键（选中项在播放头处分割）
- 框选：空白处按下拖出矩形，命中条块加入选择
- 轨道头：隐藏(眼睛)/静音(喇叭)切换 → updateUndoable
- 吸附开关：Shift+M 切换 + 状态提示
